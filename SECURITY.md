# Security and privacy

Health data and credentials are sensitive. Report vulnerabilities through GitHub private vulnerability reporting instead of a public issue.

The cross-platform core is offline and has no runtime dependencies. Vendor connectors may contact their documented services. Real records, OAuth tokens, Xiaomi sessions, QR codes, and local stores are excluded by `.gitignore`; verify staged files before every commit. This project is not medical software and must not be used for diagnosis or emergency decisions.
