# Changelog

## 0.2.0 - 2026-07-13

- Added a cross-platform, zero-dependency `healthsync` core and CLI.
- Added generic CSV import, idempotent local merge, synthetic demo generation, and a portable HTML dashboard.
- Added Linux and Windows CI plus offline tests.
- Repositioned Xiaomi and Strava integrations as optional connectors.
- Added an explicit hardware/connector status matrix and no-hardware quick start.
- Hardened ISO date validation and embedded dashboard JSON against script-tag termination.
- Added community, privacy, roadmap, release, and contribution documentation.

## 0.1.0 - 2026-06-30

- Initial Strava and Xiaomi body-composition synchronization scripts.
